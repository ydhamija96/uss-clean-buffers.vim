                                                *:CloseHiddenBuffers*
:CloseHiddenBuffers[!]      Closes all hidden buffers that don't have unsaved changes. If ! is used,
                            also close hidden terminal buffers.

                                                *:CLOSEHIDDENBUFFERS*
:CLOSEHIDDENBUFFERS[!]      Equivalent to |:CloseHiddenBuffers|.
                            This capitalized command is defined so that an abbreviated version can more 
                            easily be typed, like ":CLO" (see |user-cmd-ambiguous|).
